Anim-Num

全称: animate-number, 具有动画能力的数字. 默认采用 transform + transition 过渡动画

动画队列

当前动画未结束时, 执行了 update() 后, 会将变化的值存入一个动画队列, 同时返回 Promise 对象, 可以用来对指定的update()动画结束后的操作.

Example

有符号数字变化

<div id="demo"></div>
<script>
  var animNum = new AnimNum('#demo', 500, {
    unsigned: false
  });
  animNum.start().then(() => 
    animNum.update(1000)
  ).then(() => {
    animNum.update(-9999)
  });
</script>
